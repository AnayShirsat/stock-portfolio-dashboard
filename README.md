# Stock Portfolio Dashboard

## Overview

An interactive Stock Portfolio Dashboard built using Python and Streamlit. The application allows users to track stock investments, monitor portfolio performance, and visualize portfolio allocation using real-time market data.

## Features

* Add and remove stocks from portfolio
* Live stock price updates using Yahoo Finance
* Portfolio Value calculation
* Total Invested Amount tracking
* Profit/Loss calculation
* Return Percentage analysis
* Portfolio Allocation Pie Chart
* Profit by Stock Bar Chart
* CSV data persistence
* Download portfolio as CSV
* Streamlit web interface

## Technologies Used

* Python
* Streamlit
* Pandas
* Matplotlib
* Yahoo Finance (yfinance)
* Git & GitHub
